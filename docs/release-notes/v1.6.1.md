# Plöttr v1.6.1 — release notes

_Released 2026-05-16._

A documentation and archival-metadata release on top of v1.6.0. The
plotting tools, statistics kernels, and exports behave exactly as they
did in v1.6.0 — nothing in the app changes.

## ✨ Added

**A verified `References` section in the README.** Plöttr's statistics,
numerics, and methodology rest on a body of published research — Welch's
test, the studentized-range distribution, the Lanczos gamma
approximation, the hierarchical-clustering update formulae, and more. The
README now collects those sources in one `References` section, grouped by
area (effect sizes, test-selection methodology, numerical methods,
clustering, …). Every entry was checked against Crossref, PubMed, or the
publisher's own page for a correct author list, year, and DOI or
authoritative URL; works genuinely without a DOI are marked as such
rather than given a fabricated identifier.

That audit also pinned down the citation behind the Group Plot decision
trace. Its k ≥ 3 "Welch's ANOVA" recommendation is grounded in **Delacre,
Leys, Mora & Lakens (2019)**, _"Taking Parametric Assumptions Seriously:
Arguments for the Use of Welch's F-test instead of the Classical F-test
in One-Way ANOVA"_ ([10.5334/irsp.198](https://doi.org/10.5334/irsp.198))
— the one-way-ANOVA paper, not the 2017 Delacre _t_-test paper. The
in-app trace string is unchanged from v1.6.0; the README `References`
entry now matches it.

**`CITATION.cff` and `.zenodo.json` for Zenodo archival.** A
`CITATION.cff` file lets GitHub render a "Cite this repository" widget
and gives downstream users a ready-made citation; `.zenodo.json` supplies
the metadata Zenodo uses when it mints an archival DOI for a tagged
release. Together they make Plöttr formally citable in a methods section.
