How Plöttr handles your data, in one diagram

Your data is safe

When you drop a file in, the chart and the statistics are built right there, inside your browser tab. Plöttr has no servers.

No monitoring

No analytics, no cookies, no trackers. The page doesn't record what you click, what you upload, or how long you stayed.

Open to scrutiny

Plöttr is open source. The whole codebase — including this page — is on GitHub. Read it, fork it, or run a local copy.

Want zero network at all? Plöttr is a static site, so you can clone it once and run it forever offline:

git clone https://github.com/evompmi/plottr.git
cd plottr && open index.html  ·  (or any file:// opener)

That's it. No build step, no install. Every plot tool works the same way running from your own filesystem as it does from a server.
